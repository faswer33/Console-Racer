🏁 Console Racer – гоночная аркада в консоли

**Динамичная гоночная игра** на **7 языках программирования**.  
Управляйте автомобилем, уворачиваясь от препятствий на бесконечной трассе. Игра предлагает реалистичную консольную анимацию, счет, рекорды и сложность, возрастающую со временем.

---

## 🎮 Геймплей

- Автомобиль (символ `▲`) движется по трёхполосной дороге.
- Препятствия (символ `█`) появляются сверху и движутся вниз.
- **Цель** – проехать как можно дальше, не врезавшись.
- **Управление**: `←` / `→` или `A` / `D` – смещение влево/вправо.
- **Дополнительно**: `P` – пауза, `R` – рестарт, `Q` – выход.
- Счёт увеличивается с каждым успешным шагом; скорость растёт каждые 10 очков.
- Рекорд сохраняется в файл и загружается при старте.

## 🚀 Запуск

| Язык       | Файл         | Команда запуска                              | Зависимости |
|------------|--------------|----------------------------------------------|-------------|
| Python     | `racer.py`   | `python racer.py`                            | `keyboard`  |
| JavaScript | `racer.js`   | `node racer.js`                              | `keypress`  |

| Go         | `racer.go`   | `go run racer.go`                            | `keyboard`  |
| Rust       | `racer.rs`   | `cargo run`                                  | `crossterm` |

Все зависимости указываются в README и (для Go/Rust) в файлах конфигурации.

## ✨ Продвинутые функции (индивидуальные)

- **Python** – неблокирующий ввод с помощью `keyboard`, асинхронное обновление экрана.
- **JavaScript** – событийная модель, плавная анимация с `setInterval`.
- **Java** – использование библиотеки `jline` для обработки клавиш без Enter, многопоточность для таймера.
- **C++** – поддержка как Windows (`conio`), так и Unix (`ncurses`), условная компиляция.
- **C#** – использование `Console.ReadKey(true)` для мгновенного ввода, LINQ для работы с препятствиями.
- **Go** – горутины для одновременного чтения клавиш и обновления экрана.
- **Rust** – использование `crossterm` для кросс-платформенного ввода и цветного вывода, безопасная работа с памятью.
